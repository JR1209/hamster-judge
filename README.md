## 📖 使用方法

### 模拟模式（无需配置）

1. 在左侧边栏选择"模拟模式"
2. 输入甲方和乙方的陈述
3. 点击"提交裁决"
4. 查看结果

### AI模式（需要API密钥）

#### 使用通义千问（推荐）

1. 访问 [阿里云控制台](https://dashscope.console.aliyun.com/apiKey) 获取API Key
2. 在应用左侧边栏：
   - 选择"AI模式"
   - API提供商选择"通义千问 (Qwen)"
   - 输入API Key
   - 选择模型（推荐 qwen-plus）
3. 输入双方陈述，提交裁决

#### 使用OpenAI

1. 从 [OpenAI平台](https://platform.openai.com/api-keys) 获取API Key
2. 在应用中选择"OpenAI"作为提供商
3. 输入API Key和模型名称

#### 使用其他API

选择"自定义"并填入相应的API信息。

## ⚙️ 自定义判断标准

在侧边栏勾选"使用自定义判断标准"，可以自定义AI的评判依据。

**示例：**

```
请根据以下标准进行评判：
1. 沟通有效性（30%）：表达是否清晰、理性
2. 情感合理性（25%）：诉求是否合情合理
3. 责任意识（25%）：是否愿意承担责任和改进
4. 尊重程度（20%）：对对方的尊重和理解程度

请给出：
- 甲方得分（0-100）
- 乙方得分（0-100）
- 详细分析
- 建议
```

更多配置说明请查看 [配置说明.md](配置说明.md)
