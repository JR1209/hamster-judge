# 📸 使用演示和示例

## 🎬 使用流程演示

### 模拟模式使用流程

1. **打开应用**
   - 看到标题：⚖️ 仓鼠法官裁决系统 🐹
   - 仓鼠法官在线待命

2. **侧边栏设置**
   - 选择"模拟模式"（默认）
   - 无需其他配置

3. **输入陈述**
   - 左侧：甲方陈述
   - 右侧：乙方陈述

4. **提交裁决**
   - 点击"⚖️ 提交裁决"按钮
   - 等待仓鼠法官审理

5. **查看结果**
   - 看到胜负判定
   - 查看百分比和进度条
   - 阅读详细裁决书

### AI模式使用流程

1. **配置API**
   - 侧边栏选择"AI模式"
   - 选择API提供商（推荐：通义千问）
   - 输入API Key
   - 选择模型

2. **（可选）自定义标准**
   - 勾选"使用自定义判断标准"
   - 输入你的评判依据

3. **输入陈述并提交**
   - 同模拟模式

4. **查看AI分析**
   - 看到AI详细分析
   - 专业的建议和评分

## 📝 实际案例演示

### 案例1：游戏时间分配

**背景：** 男生玩游戏时间过长，女生感到被忽视

**甲方陈述：**
```
我真的很生气。他每天下班回家第一件事就是打开电脑玩游戏。
我想和他聊聊今天发生的事情，他总是说"等我打完这一局"。
但一局接一局，根本停不下来。我们已经好几天没有好好说话了。
我感觉在他心里，游戏比我重要。我不是不让他玩，
但能不能至少先陪我聊15分钟再玩？我也需要陪伴啊。
```

**乙方陈述：**
```
我承认我玩游戏时间是有点多，但我也不是整天都在玩啊。
我每天上班压力很大，游戏是我唯一的放松方式。
而且我也没有不理她，我会回应她说的话，只是可能反应慢了点。
周末我也会陪她出去玩啊。我觉得她对我要求太严格了，
我也需要自己的空间和时间，难道下班后连放松的权利都没有了吗？
```

**AI裁决结果示例：**

```
【甲方得分】：65分
【乙方得分】：58分

【详细分析】
甲方在沟通中表达了明确的情感需求（陪伴15分钟），
这是一个合理且具体的诉求。她也表明了并非完全反对游戏，
只是希望有优先级。表达相对理性克制。

乙方承认了问题的存在，这是积极的一面。但在论述中
更多强调了自己的需求（压力、放松），对伴侣的情感需求
理解不够深入。"反应慢了点"显示出对问题严重性认知不足。

【调解建议】
1. 建立"陪伴优先15分钟"规则：下班回家先陪伴15分钟
2. 设定游戏时间段：如晚上8-10点为游戏时间
3. 质量陪伴：陪伴时放下手机，专注倾听
4. 周末约会：确保每周至少一次高质量约会
5. 相互理解：女生也给予男生合理的放松空间
```

### 案例2：家务分配问题

**背景：** 家务分配不均

**甲方陈述：**
```
我真的累了。每天下班回家，我要做饭、洗碗、拖地、洗衣服。
他呢？坐在沙发上刷手机。我们都是上班族，为什么家务都是我做？
我提醒他帮忙，他说"好的好的"，然后继续刷手机。
我不想当他妈妈，一直催他做事。我也想下班后休息啊！
```

**乙方陈述：**
```
我不是不做家务，我周末会打扫卫生，垃圾都是我倒的，
家里东西坏了也是我修的。她总是看不到我做的事情，
只记得我没做的。而且她做家务的标准太高了，
我做了她还要挑剔。既然怎么做都不满意，那我还不如不做。
```

**AI裁决结果示例：**

```
【甲方得分】：72分
【乙方得分】：45分

【详细分析】
甲方的诉求有充分的事实基础（具体列举了多项家务），
表达了公平分担的合理期待。情绪虽然激动但诉求明确。

乙方存在几个问题：1)将周末家务和日常家务对等，
忽视了日常家务的频率；2)"既然怎么做都不满意，那我还不如不做"
显示出逃避责任的态度；3)缺乏主动承担的意识。

【调解建议】
1. 明确家务清单：列出所有日常和周末家务
2. 协商分工：根据各自时间和能力分配任务
3. 建立轮换制：某些家务可以按周轮换
4. 降低标准：接受"做了"比"完美"更重要
5. 定期检视：每月讨论一次，及时调整
```

### 案例3：社交媒体矛盾

**背景：** 一方在社交媒体上的行为引起不满

**甲方陈述：**
```
他的朋友圈从来不发我们的合照，好像我不存在一样。
但他会发和朋友聚会的照片，会发风景，就是不发我。
我问他为什么，他说"没必要秀恩爱"。可是我不是要秀恩爱，
我只是希望他能让别人知道他有女朋友。这让我感觉他是不是
不想承认我们的关系？我很没有安全感。
```

**乙方陈述：**
```
我觉得感情是两个人的事，不需要在社交媒体上展示给别人看。
我对她很好，这是最重要的。朋友圈发不发照片不代表什么。
而且我不喜欢秀恩爱，觉得很肉麻。她为什么这么在意别人的看法？
我们的感情好不好，不是别人说了算的。
```

**AI裁决结果示例：**

```
【甲方得分】：60分
【乙方得分】：55分

【详细分析】
这是一个典型的安全感和价值观冲突案例。

甲方的诉求背后是对关系确认的需要，这是正常的情感需求。
但可能过度依赖外部认可来获得安全感。

乙方强调私密性有其合理性，但"不需要向别人证明"的论点
忽视了伴侣的情感需求。完全拒绝而非寻求妥协，显示出缺乏
对伴侣感受的重视。

【调解建议】
1. 寻找平衡：偶尔发一些合照（如节日、纪念日）
2. 线下确认：在朋友聚会时大方介绍女友
3. 其他方式：通过其他方式给予安全感（承诺、行动）
4. 沟通需求：女生说明为什么这对她重要
5. 建立信任：通过日常行为建立更深的信任感
```

## 🎯 不同场景的判断标准示例

```
请根据以下标准评判：
1. 表达清晰度（25%）：是否清楚说明问题
2. 情绪控制（20%）：表达是否理性
3. 倾听意愿（25%）：是否理解对方观点
4. 解决方案（30%）：是否提出建设性建议

请客观评分并提供沟通改进建议。
```

### 场景B：责任分配

```
请根据以下标准评判：
1. 事实依据（35%）：陈述是否有具体事实
2. 公平意识（30%）：是否考虑双方的付出
3. 承担意愿（25%）：是否愿意承担责任
4. 实际行动（10%）：是否有实际改进行动

请给出公平的评分和具体建议。
```

### 场景C：情感需求

请作为情感咨询师评判：

3. 共情能力（25%）：是否理解伴侣感受
4. 妥协意愿（20%）：是否愿意为关系妥协

请温柔地给出建议，帮助双方理解彼此。

## 💡 使用技巧

### 获得更好结果的技巧

1. **详细陈述**
   - 提供具体事例
   - 说明时间、频率
   - 描述自己的感受
   - 说明自己的诉求

2. **理性表达**
   - 避免纯粹情绪宣泄
   - 用事实支撑观点
   - 承认自己的不足
   - 表达改进意愿

3. **双方参与**
   - 都要认真写陈述
   - 诚实表达真实想法
   - 不要互相攻击
   - 尊重对方观点

4. **自定义标准**
   - 根据实际情况调整权重
   - 明确你最看重的方面
   - 要求具体的建议
   - 设定合适的语气（温和/严格）

### 注意事项

⚠️ **这不是真正的法庭**
- 结果仅供参考
- 重要的是沟通和理解
- 不要把结果当作"判决"
- 关系需要双方共同努力

⚠️ **适用范围**
- 适合：日常小矛盾、意见分歧
- 不适合：严重冲突、原则性问题
- 娱乐为主，辅助沟通

⚠️ **如果出现严重问题**
- 请寻求专业心理咨询
- 或婚姻家庭咨询师帮助
- 本应用不能替代专业咨询
