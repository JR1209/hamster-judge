# 🚀 开始使用仓鼠法官

选择你的使用方式：

## 方式1️⃣：直接访问在线版本（最简单）

如果已经部署到云端，直接打开网址：
```
https://你的应用名.streamlit.app/
```

## 方式2️⃣：在本地运行

### Mac/Linux 用户

打开终端，运行：
```bash
cd /Users/z/Desktop/仓鼠法官
pip install -r requirements.txt
streamlit run hamster_judge.py
```

### Windows 用户

打开命令提示符（CMD），运行：
```bash
cd C:\Users\你的用户名\Desktop\仓鼠法官
pip install -r requirements.txt
streamlit run hamster_judge.py
```

然后浏览器会自动打开，或手动访问：http://localhost:8501

## 方式3️⃣：部署到云端

### 最快方法（Mac/Linux）

```bash
cd /Users/z/Desktop/仓鼠法官
bash deploy.sh
```

按照提示操作即可！
