## ⚙️ 快速配置指南

### 使用模拟模式（无需配置）
1. 运行程序
2. 在侧边栏选择"模拟模式"
3. 开始使用！

### 使用AI模式（需要API Key）

#### 获取通义千问API Key：
1. 访问：https://dashscope.console.aliyun.com/apiKey
2. 注册/登录阿里云账号
3. 创建API Key
4. 复制密钥

#### 在应用中配置：
1. 运行程序
2. 侧边栏选择"AI模式"
3. API提供商选"通义千问 (Qwen)"
4. 输入API Key
5. 模型选择：qwen-plus（推荐）
6. 开始使用！

## 🆘 遇到问题？

### 常见问题解决

1. **模块找不到**
   - 检查 requirements.txt 是否完整
   - 重新运行 `pip install -r requirements.txt`

2. **API调用失败**
   - 检查API Key是否正确
   - 检查API余额
   - 检查网络连接

3. **部署失败**
   - 查看Streamlit Cloud的日志
   - 检查代码是否有语法错误
   - 确认requirements.txt正确

4. **应用访问慢**
   - Streamlit免费版资源有限，正常现象
   - 可以优化代码
   - 或考虑升级付费版
